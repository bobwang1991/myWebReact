# react&reactNative

目录：

     shDemo:搜狐新闻的组件

react安装：

1、下载安装node		https://nodejs.org/en/download/

2、安装yeoman	npm install -g yo （yo --version  检测版本）

3、npm install -g generator-react-webpack

4、创建文件夹和进入文件夹	md firstTime  ； cd firstTime

5、搭建项目文件并安装	yo react-webpack firstTime

react运行：
   npm start

如果已经搭建好的但是没安装执行：
    npm install
